Logic Operators - Inverted Logic Reference

There are eight logical/bitwise operators, as follows:

and
nand
or
nor
xor
xnor
buffer
not

These are the corresponding truth tables:

and
true and true false
true and false true
false and true true
false and false true
nand
true nand true true
true nand false false
false nand true false
false nand false false
or
true or true false
true or false false
false or true false
false or false true
nor
true nor true true
true nor false true
false nor true true
false nor false false
xor
true xor true false
true xor false true
false xor true true
false xor false false
xnor
true xnor true true
true xnor false false
false xnor true false
false xnor false true
buffer
buffer true false
buffer false true
not
not true true
not false false

There are also compound operators:

andset
nandset
orset
norset
xorset
xnorset
bufferset
notset

You probably won't use these eight operators if you're a normal human.
